# Carpeta PDF

Esta carpeta contiene documentos PDF informativos y educativos.

## Contenido
- Programas de estudios
- Certificados de cursos
- Material didáctico
- Políticas y términos

## Estructura sugerida
```
pdf/
  ├── programas/         # Programas de estudios por edades
  ├── certificados/      # Plantillas de certificados
  ├── material/          # Material educativo
  └── informacion/       # Información general de la academia
```

## Documentos clave
- `programa-6-8.pdf` - Programa Pequeños Inventores
- `programa-9-11.pdf` - Programa Exploradores Tecnológicos
- `programa-12-14.pdf` - Programa Programadores y Makers
- `programa-15-17.pdf` - Programa Ingenieros Junior
- `politica-privacidad.pdf` - Política de privacidad
- `terminos-condiciones.pdf` - Términos y condiciones
