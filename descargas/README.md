# Carpeta Descargas

Esta carpeta contiene archivos descargables para los estudiantes y padres.

## Tipos de archivos
- `.zip` - Archivos comprimidos con proyectos
- `.pdf` - Guías y documentación
- `.exe` - Instaladores de software educativo
- `.apk` - Aplicaciones para Android

## Estructura sugerida
```
descargas/
  ├── guias/           # Guías y manuales en PDF
  ├── software/        # Instaladores y herramientas
  ├── proyectos/       # Proyectos de ejemplo comprimidos
  └── recursos/        # Material educativo adicional
```

## Uso en HTML
```html
<a href="descargas/archivo.zip" download="nombre-archivo.zip">
  Descargar Proyecto
</a>
```

## Contenido sugerido
- Instaladores de Arduino IDE
- Proyectos de Scratch exportados
- Guías de LEGO Mindstorms
- Tutoriales de Python
- Plantillas de proyectos ROS2
