Información para las Familias

Resolvemos vuestras dudas sobre el programa más avanzado

❓

¿Es necesaria experiencia previa?

Recomendamos conocimientos de Python básico y programación. Realizamos una entrevista técnica previa para evaluar el nivel y asegurar que el alumno pueda aprovechar al máximo el programa. Si viene del programa 12-14 años está perfectamente preparado.

💻

¿Qué equipo necesita?

Ordenador portátil propio (recomendado mínimo 8GB RAM). Incluimos Raspberry Pi para proyectos, acceso a robots de la academia y todas las licencias de software necesarias. Linux es preferible pero pueden trabajar en Windows/Mac.

🏆

¿Pueden participar en competiciones?

¡Sí! Preparamos específicamente para First Robotics Competition (FRC), World Robot Olympiad (WRO) y olimpiadas de programación. Los alumnos forman equipos y representan a la academia en competiciones nacionales e internacionales.

Ejemplos de lo que Crean

Proyectos de nivel profesional que impresionan en universidades

01
🤖

Robot Autónomo con SLAM

Sistema completo de navegación autónoma usando ROS2, LIDAR y algoritmos de path planning.

Tecnologías:

  • ROS2 Humble + Nav2 stack
  • Cartographer para SLAM
  • Path planning con A* optimizado
  • Obstacle avoidance en tiempo real
  • Visualización en RViz
Resultado: Robot que mapea entornos y navega autónomamente evitando obstáculos
02
🧠

Sistema de Visión Artificial con IA

Detector de objetos en tiempo real con red neuronal convolucional entrenada desde cero.

Pipeline completo:

  1. Recolección y etiquetado de dataset
  2. Arquitectura CNN custom con TensorFlow
  3. Data augmentation y training
  4. Optimización para tiempo real
  5. Integración con OpenCV y ROS
  6. Deploy en Raspberry Pi
Aplicación: Detección de objetos a 30 FPS para robótica de competición
03
🏆

Proyecto First Robotics

Robot completo para competición FRC con visión, manipulador y estrategia autónoma.

Componentes del sistema:

  • Control de manipulador con cinemática inversa
  • Visión artificial para tracking de objetivos
  • Sistema de navegación campo automático
  • Telemetría y dashboard en tiempo real
  • Estrategia autónoma con IA
  • Documentación técnica completa
Logro: 2º puesto nacional First Robotics 2025

Preguntas Frecuentes

Resolvemos tus últimas dudas sobre el programa

❓

¿Qué nivel de programación necesito para empezar?

Recomendamos conocimientos básicos de Python (variables, bucles, funciones). Realizamos una entrevista técnica previa para evaluar el nivel. Si has completado nuestro programa 12-14 años, estás perfectamente preparado. También aceptamos alumnos con experiencia autodidacta demostrable.

💻

¿Necesito ordenador potente o sistema operativo específico?

Recomendamos portátil con mínimo 8GB RAM (preferible 16GB) y 256GB almacenamiento. Linux Ubuntu es lo ideal pero también trabajamos con Windows/Mac usando máquinas virtuales o WSL. Te ayudamos con la configuración en la primera sesión.

🏆

¿Realmente preparan para competiciones como First Robotics?

¡Absolutamente! El programa está diseñado pensando en competiciones. Nuestros alumnos han conseguido 2º puesto nacional en FRC y múltiples premios en WRO. Dedicamos sesiones específicas a preparación de competiciones y formamos equipos para representar a la academia.

🎓

¿Cómo ayuda esto para entrar en la universidad?

Enormemente. Los alumnos salen con portfolio profesional en GitHub, experiencia con tecnologías industriales (ROS), proyectos de IA documentados y certificado de nivel avanzado. Además, los logros en competiciones se valoran mucho en procesos de admisión a ingenierías top. Proporcionamos cartas de recomendación.

⏰

¿Cuánto tiempo debo dedicar fuera de clase?

Los proyectos requieren trabajo fuera de clase. Recomendamos 2-4 horas semanales adicionales, especialmente si se preparan para competiciones. Es un compromiso serio, similar a una asignatura universitaria, pero el resultado vale la pena.

💰

¿Qué incluye exactamente el precio?

95€/mes incluye: sesiones de 2h semanales, kit Raspberry Pi (tuyo), acceso ilimitado a robots de la academia, todas las licencias software (TensorFlow, ROS, etc.), mentorización personalizada, hosting GitHub Pro, certificado final y preparación para competiciones. Solo pagas matrícula una vez (50€).
