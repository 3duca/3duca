¿Por qué este programa?

En la era digital, entender tecnología es fundamental. Este curso no solo enseña a programar, sino a pensar como ingenieros, resolver problemas complejos y crear soluciones tecnológicas reales.

Profesiones más demandadas: Desarrollo software, IA, Data Science, IoT, Robótica (>30K€/año iniciales)

📚

¿Qué nivel previo necesitan?

Idealmente, conocimientos básicos de programación (Scratch, Python básico). Si no tienen experiencia, ofrecemos curso de nivelación de 4 semanas o pueden empezar en programa 9-11 años.

Ejemplos de lo que Crean

Proyectos reales de nuestros makers de 12-14 años

01
🏠

Sistema Domótica IoT

Control total de una habitación mediante app móvil propia: iluminación, temperatura, alertas de seguridad y IA predictiva que aprende horarios del usuario.

Tecnologías usadas:

  • ESP32 con WiFi y sensores
  • App móvil MIT App Inventor
  • Machine Learning básico
  • Protocolo MQTT para IoT
Resultado: Sistema completo hardware + software + IA funcional
02
🌡️

Predictor Meteorológico con IA

Estación meteorológica Arduino que recopila datos en tiempo real y predice el clima usando Random Forest. Precisión del 87% en predicciones a 24 horas.

Pasos del proyecto:

  1. Construir estación con sensores Arduino
  2. Recopilar datos durante semanas
  3. Entrenar modelo ML con Python
  4. Desplegar en ThingSpeak cloud
  5. Crear dashboard de visualización
Lo mejor: ¡Predicciones más precisas que algunas apps comerciales!
03
🤖

Robot Explorador con Visión

Robot autónomo con cámara y visión por computadora que mapea entornos, evita obstáculos y optimiza rutas. Control remoto mediante app móvil.

Características:

  • Arduino Mega + sensores ultrasónicos
  • Cámara con OpenCV básico
  • Comunicación Bluetooth
  • Algoritmos de pathfinding
  • App de control y visualización
Creatividad: Cada alumno personaliza funcionalidades únicas

Preguntas Frecuentes

Resolvemos tus últimas dudas

📚

¿Qué nivel previo necesito?

Idealmente conocimientos básicos de Scratch o programación visual. Si no tienes experiencia, ofrecemos curso de nivelación de 4 semanas incluido sin coste adicional si te inscribes antes de agosto.

🎁

¿Qué incluye el kit Arduino?

Arduino UNO original, breadboard, 30+ sensores (temperatura, movimiento, distancia, luz), motores, LEDs, cables, resistencias y todos los componentes necesarios. Todo es tuyo para llevarlo a casa.

📊

¿Cómo se evalúa el progreso?

Sin exámenes. Evaluación continua por proyectos completados. Informe trimestral con habilidades técnicas específicas, acceso a repositorio GitHub del alumno y presentación de proyectos cada trimestre.

📅

¿Qué pasa si me pierdo una clase?

No hay problema. Cada proyecto dura varias sesiones, por lo que puedes recuperar fácilmente. Además, todos los materiales están disponibles en GitHub para consulta.

🚀

¿Puedo continuar después de este curso?

Sí, el 95% de alumnos continúan al programa 15-17 años (nivel avanzado) donde trabajarán con ROS, Deep Learning y robótica profesional. También pueden participar en competiciones como FLL, WRO o FTC.

💡

¿Necesito comprar material adicional?

No. El kit Arduino y todos los componentes están incluidos en la matrícula. Los componentes consumibles se reponen sin coste durante todo el curso.
